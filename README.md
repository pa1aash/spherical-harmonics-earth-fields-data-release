# Spherical-harmonic truncation of planetary potential fields

This repository provides all code and derived results needed to reproduce the paper:

> Palaash Gang (2026). "A reproducible framework for measuring the compressibility of truncated spherical-harmonic models of planetary potential fields."

The pipeline characterises how the relative L²(S²) reconstruction error `eps(l_max)` decays as a function of truncation degree for Earth's gravity field (EGM2008), Earth's main magnetic field (IGRF-13), Earth's crustal magnetic field (NGDC-720), and the gravity fields of the Moon (GRGM1200B) and Mars (GMM3). Key outputs are an empirical decay exponent measured by OLS regression, threshold degrees at 10/5/1% error, a rate-distortion / codec bit-efficiency analysis, and temporal / planetary / functional comparisons. All model coefficients are fetched on first run via `pyshtools`' built-in dataset API and cached locally; subsequent runs need no network access.

## Installation

**Recommended: conda (pyshtools 4.14.1 must be installed via pip into the conda env)**

```bash
conda env create -f environment.yml
conda activate shtrunc
pip install -e .
```

**Alternative: pip only (linux/macOS)**

```bash
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
pip install -e .
```

## Reproduce

| Command | Effect |
|---|---|
| `make test` | Run the full test suite (139 tests; no downloads) |
| `make all` | Create venv, install deps, run the full pipeline (downloads models on first run) |
| `make figures` | Regenerate figures 1–5 and A/B/D (assumes deps installed, models cached) |
| `make figures-cache` | Rebuild all figures from committed `results/` artifacts; no computation or downloads |
| `make audit` | Independent re-derivation from raw coefficients; prints PASS/FAIL per check |
| `make aux` | Regenerate the auxiliary result tables that feed Figures A, B, D |

The first `make all` / `make figures` downloads model files into `~/.cache/pyshtools/`: EGM2008 (~250 MB, ICGEM), IGRF-13 (~40 kB, NOAA NCEI), NGDC-720 V3 (~3 MB, NOAA NCEI), Moon GRGM1200B (~large, GSFC), Mars GMM3 (JPL). After that, everything runs offline.

`make test` needs only the installed packages and `results/fit.json`; it does not download any model files.

## Results artifacts (in `results/`)

| File | Description |
|---|---|
| `epsilon_curves.csv` | Full `eps(l_max)` curves for all fields (Parseval + grid-based values, coefficient counts) |
| `thresholds.csv` | `l_max` and coefficient counts at 10/5/1% error per field |
| `fit.json` | Decay exponents ± jackknife SE, spectral index α ± WLS SE, reduced χ², software versions |
| `summary.txt` | Seed, data versions, and all headline numbers in one place |
| `planetary.csv` | Planetary geoid truncation (Earth / Moon / Mars), one row per body |
| `planetary_curves.csv` | Full eps(l_max) curves for Earth/Moon/Mars at several epsilon levels |
| `functionals.csv` | Gravity functional comparison (geoid / disturbance / anomaly / Trr) |
| `temporal_igrf.csv` | IGRF temporal sweep 1900–2025: per-epoch slopes and threshold degrees |
| `bandlimit.csv` | EGM2008 signal-vs-noise crossover (effective bandlimit Gate C result) |
| `water_fill.json` | Rate-distortion floor (water-filling) vs float32 bits, per field and epsilon |
| `codec_gap.json` | Greedy and entropy codec bits vs RD floor, per field and epsilon |
| `radius_compressibility.csv` | IGRF eps slope vs radius surface→CMB (downward-continuation analysis) |
| `anisotropy.csv` | Best-N-term vs SH truncation gap (Gate: SH ≈ best-N-term for geoid) |
| `wavelet_comparison.json` | SH vs wavelet bit-efficiency at matched patch epsilon (Gate D result) |
| `resolution.csv` | SH half-wavelength resolution per degree (used in appendix table) |
| `verification.json` | Parseval cross-check micro-numbers cited in the Verification paragraph |
| `numbers.tex` | LaTeX macros for all headline numbers (auto-generated by `make_figures.py`) |

## Data provenance

All model coefficients are fetched via **pyshtools 4.14.1** built-in datasets. Raw files are cached in `~/.cache/pyshtools/` and never vendored in this repository.

| Model | pyshtools accessor | Source institution | Reference |
|---|---|---|---|
| EGM2008 (gravity, degree 360) | `pyshtools.datasets.Earth.EGM2008(lmax=360)` | ICGEM, GFZ Potsdam | Pavlis et al. (2012), *JGR Solid Earth* 117, B04406 |
| IGRF-13 (magnetic main field, degree 13) | `pyshtools.datasets.Earth.IGRF_13(lmax=13, year=2020.0)` | NOAA NCEI | Alken et al. (2021), *Earth Planets Space* 73, 49 |
| NGDC-720 V3 (crustal magnetic, degree 133) | `pyshtools.datasets.Earth.NGDC_720_V3(lmax=133)` | NOAA NCEI | Maus (2010), *G³* 11, Q06015 |
| Moon GRGM1200B (GRAIL, degree 360) | `pyshtools.datasets.Moon.GRGM1200B(lmax=360)` | GSFC | Goossens et al. (2020), *JGR Planets* 125, e2019JE006086 |
| Mars GMM3 (degree 90) | `pyshtools.datasets.Mars.GMM3(lmax=90)` | JPL | Genova et al. (2016), *Icarus* 272, 228–245 |

Reference ellipsoids: `boule.WGS84` (Earth), `boule.Moon2015` (Moon), `boule.Mars2009` (Mars).
